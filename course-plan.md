# Programación Web – Módulo Fundamental

## 1. Resumen del curso

Este curso de aproximadamente **1 hora** introduce a estudiantes de secundaria a los conceptos fundamentales del **desarrollo web**, utilizando solo tecnologías de frontend: **HTML** y **CSS**.  
Está pensado para que cualquier estudiante, incluso sin experiencia previa, pueda crear su primera página web desde el navegador, sin instalar programas.

- **Duración total:** ~60 minutos  
- **Público objetivo:** Estudiantes de 12 a 17 años sin experiencia previa en programación  
- **Prerrequisitos:** Ninguno  
- **Herramientas necesarias:**  
  - Navegador web (Chrome, Firefox, Edge, Safari)  
  - Acceso a Internet  
  - Editor online: **CodePen**

- **Repositorio del curso:**  
  https://github.com/2025-2-AplicacionesWeb-DalComp/webdev-course-DaL-Company.git

## 2. Secuencia de sesiones

### Sesión 1 — Conceptos iniciales del desarrollo web (10 min)

**Descripción:**  
Introducción al mundo del desarrollo web: qué es una página web, qué partes la componen y cuál es el rol de tecnologías como **HTML** y **CSS**.
Se explica la diferencia entre **frontend** y **backend**, y se presenta CodePen como entorno online de práctica.

**Objetivos específicos:**

- Identificar qué es un sitio web.  
- Comprender el rol de HTML y CSS. 
- Reconocer cómo el navegador interpreta un documento.  

**Recurso audiovisual:**  
- Video: https://youtu.be/06TaPoQSI7U

**Entorno de práctica:**  
- CodePen base:
  * Ejercicio 1: https://codepen.io/uszhiyts-the-looper/pen/pvyNZMg
  * Ejercicio 2: https://codepen.io/uszhiyts-the-looper/pen/gbrLdYE

**Ejercicios:**

- **Ejercicio 1:**  
  Crear una mini página con un `<h1>` y dos párrafos `<p>` describiendo al estudiante.  

- **Ejercicio 2:**  
  Probar diferentes niveles de encabezados (`<h1>`–`<h4>`) y un párrafo explicando la diferencia visual.

**Mini reto:**  
- Crear una mini tarjeta personal con:  
  - `<h1>` → nombre  
  - `<h2>` → apellido  
  - `<p>` → breve descripción

---

### Sesión 2 — Estructura mínima de un documento HTML (12 min)

**Descripción:**  
Se explica la estructura básica de un archivo HTML, el significado de `<!DOCTYPE html>`, el uso de `<html>`, `<head>` y `<body>`, y la importancia del título de la página.

**Puntos clave:**

- Estructura mínima de un documento HTML.  
- Diferencia entre `<head>` y `<body>`.  
- Importancia del atributo `lang` y del `<meta charset="UTF-8">`.  

**Recurso audiovisual:**  
- Video: https://youtu.be/0MQ6oBm2TCA

**Entorno de práctica:**  
- CodePen base: https://codepen.io/uszhiyts-the-looper/pen/NPNbLqW

**Ejercicio principal:**  

- Completar un archivo con el esqueleto HTML mínimo:
- Agregar un título descriptivo en `<title>`.  
- Escribir un mensaje de bienvenida dentro del `<body>`.

**Mini reto:**  

1. Crear un archivo llamado `index.html`.  
2. Pegar el esqueleto mínimo de HTML.  
3. Cambiar el `<title>` por el nombre del estudiante.  
4. Dentro del `<body>`, escribir una breve descripción personal.

### Sesión 3 — Elementos HTML comunes (12 min)

**Descripción:**  
Se presentan elementos muy usados en páginas web: imágenes, enlaces y listas.  
Se trabaja el uso de atributos como `src`, `href` y `alt`.

**Contenido:**

- Inclusión de imágenes con `<img>`.  
- Enlaces con `<a>`.  
- Listas con `<ul>`, `<ol>` y `<li>`.  
- Uso básico de atributos (`src`, `href`, `alt`).

### Sesión 4 — Introducción a la hoja de estilos CSS (13 min)

**Descripción:**  
Se introduce **CSS** para dar estilo a las páginas. Se explica la sintaxis básica de una regla, la diferencia entre etiqueta, clase e id, y se aplican estilos a textos, fondos y contenedores.

**Aspectos desarrollados:**

- Sintaxis básica de una regla CSS (`selector { propiedad: valor; }`).  
- Estilos generales de texto (color, tamaño, alineación).  
- Backgrounds simples y con gradiente.  
- Diferencias entre clases (`.clase`) e identificadores (`#id`).  
- Vinculación de CSS desde el HTML en CodePen.  

**Recurso audiovisual:**  
- Video: https://youtu.be/gBnFrPrnhUY

**Entorno de práctica:**  
- CodePen base: https://codepen.io/uszhiyts-the-looper/pen/ZYWBMBK

**Ejercicio principal:**

- Dar estilo a una “página colorida”:  
- Fondo con color o gradiente.  
- Título `<h1>` centrado y con color.  
- Párrafos dentro de una caja con borde, padding y fondo suave.

**Mini reto:**

- Crear un botón con la clase `.boton` que cambie de color al pasar el mouse (`:hover`).  

---

### Sesión 5 — Construcción de una página web sencilla (13 min)

**Descripción:**  
Se combinan todos los conceptos vistos (HTML + CSS) para construir una página sencilla que pueda servir como mini portafolio o tarjeta personal.

**Habilidades desarrolladas:**

- Organización del contenido en secciones.  
- Aplicación simultánea de HTML y CSS.  
- Uso de encabezado, contenido principal y pie de página.  
- Integración de estilos coherentes (colores, tipografía, espaciados).  

**Recurso audiovisual:**  
- Video: 

**Entorno de práctica:**  
- CodePen base: 

**Ejercicio principal:**

- Construir una página tipo “presentación personal” que incluya:
